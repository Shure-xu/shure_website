# Page Topology

## Sections
1. Fixed floating navigation
   - Rounded, translucent, always visible.
   - Links scroll to Work, Services, Notes, and Contact.

2. Hero
   - First viewport signal with oversized Chinese headline.
   - Two-column desktop layout, stacked mobile layout.
   - Visual side includes generated workspace image and bright accent block.

3. Selected Work
   - Three-card project grid.
   - Image hover scale and arrow micro-interaction.

4. Services
   - Dark full-width band.
   - Large service rows with accent arrows.

5. Notes
   - Editorial note area with one large color block and three smaller entries.

6. Contact
   - Bright footer CTA with email link.

## About Page

1. Fixed floating navigation.
2. Editorial intro with a Chinese exhibition-style statement and short personal archive notes.
3. Five two-column practice modules, each pairing a Chinese practice name with an English label.
4. Existing personal video and the existing metal-texture card form a profile diptych.
5. Cross-media direction list.
6. Shared black `#contact` footer.

## Interaction Model
- Navigation: anchor-scroll links.
- Work cards: hover-driven image scale and arrow movement.
- Contact links: simple mailto links.
- Page-level: smooth scroll unless reduced motion is requested.
