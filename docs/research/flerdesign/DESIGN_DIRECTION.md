# Fler Design Reference Direction

## Intent
This project uses `https://www.flerdesign.com/` as a visual reference, not as a direct clone. The site borrows the reference's editorial studio feeling: large confident typography, asymmetric visual blocks, high-contrast color moments, rounded media containers, and a mix of work, services, notes, and contact sections.

## Reference Signals
- Framer-built studio site with a magazine-like homepage structure.
- Strong first viewport with bold type, image-led editorial composition, and clear navigation.
- Sections communicate studio identity, recent work, services, writing/podcast style content, and contact.
- Visual energy comes from contrast: quiet paper background, dark ink sections, bright accent blocks, and oversized headings.

## Adaptation Rules
- Do not copy Fler's logo, exact copy, project names, images, or brand assets.
- Keep the layout rhythm and creative-studio tone.
- Replace all content with personal-site placeholders that can later become the user's real name, work, services, and contact details.
- Use local generated placeholder imagery until the user provides real photos or portfolio assets.

## Current Content Model
- Hero: personal studio positioning and introductory statement.
- Work: three featured project cards.
- Services: capability list.
- Notes: blog/podcast/thinking area.
- Contact: large CTA footer.
